# Examples

- `esp32-bad-gpio-audit/`: Intentionally unsafe GPIO12 usage for pin audit demos.
- `esp32-serial-verify/`: Emits `BOOT_OK` + `HEARTBEAT` for runtime verification demos.
- `missing-library-diagnose/`: Missing header example for build diagnostics.
- `mcpu-arduino-blink/`: MCP-U integration scaffold (no third-party code included).
