# AI Agent Skills

`platformio-mcp-server` ships with specialized Agent Prompts located in this folder. These prompts are designed to vastly improve an AI's autonomous capability with hardware builds natively.

## How to Use

If you are using an AI coding assistant like **Claude**, **Cline**, or **Antigravity**, you can significantly enhance its contextual understanding of PlatformIO interactions by importing these guidelines:

1. Navigate to the `pio-manager/` directory within this folder.
2. Copy the `.md` guidelines and asset workflows.
3. Paste them into your local AI agent's custom skill or rules directory (e.g., `.agents/skills/`, `.cline/rules`, or your tool's relevant equivalent).

By bringing these specialized instructions into your agent's context, it will understand hardware compilation, flashing, and serial monitoring using the PlatformIO MCP server safely.
